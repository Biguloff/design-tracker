Что iOS НЕ рендерит из SVG

  • <foreignObject> — полностью игнорируется
  • CSS backdrop-filter: blur() — игнорируется
  • SVG <filter> / <feGaussianBlur> — игнорируется
  • <mask> — частичная поддержка (простые OK, сложные нет)
  • CSS properties через style="" — частичная поддержка

Рекомендация: для теней использовать CALayer.shadow* или ThemeEffects.shadowLevel*. Для blur — UIVisualEffectView.
