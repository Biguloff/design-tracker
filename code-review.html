5. Архитектура (Combine, CALayer) — 3/10
›

✘ 0 Combine-подписок — критический пропуск

Гайд: «Если ты используешь .cgColor (для CALayer) — нужна подписка, потому что CGColor — это не UIColor, он не обновляется автоматически»

Мы добавили 0 ThemeManager.shared.$currentTheme подписок. При этом внесли 11+ .cgColor assignments:

  • topView.layer.borderColor = cardBorder.cgColor
  • middleView.layer.borderColor = cardBorder.cgColor
  • borderLayer.strokeColor = UIColor.storiesCardBorder.cgColor
  • gradientTint.colors = [UIColor.background...cgColor]
  • skipButton.layer.borderColor = UIColor.text.cgColor

Все сломаются при runtime-смене темы. Цвет установится один раз и не обновится.

⚠ 8 type checks по конкретным классам

theme is NewDarkTheme || theme is NewLightTheme — 8 мест. При добавлении 6-й темы — 8 потенциальных багов.

Правильнее: glass.blurStyle != nil или флаг supportsGlassEffects: Bool в ThemeProtocol.

⚠ lazy var blurView — одноразовая инициализация

В Task 7 мы заменили let на lazy var чтобы получить доступ к ThemeTokenStorage. Blur style установится один раз при первом обращении и не обновится при смене темы.
