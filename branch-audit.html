📈 Масштаб миграции

  • Таймураз: 247 файлов, широкая миграция hardcoded UIColor(red:) → theme tokens по всему приложению. Охватил auth, AI chat, settings, profile, messenger.
  • Мы: ~40 файлов, глубокая работа над Figma-точностью — gradient locations, glass effects, layout tokens (sizes, gaps, padding, chatBubbleCorners), pixel-perfect значения.
  • Пересечение: ~10 файлов изменены в обеих ветках (ThemeEffects, ThemeColors, themes, MessengerVC, UIButton, AvatarView, MessageActions)
🔍 Ключевые различия в подходе
›
РазличиеНаш подходПодход Таймураза
1. Философия Figma-first: каждый токен из Figma Variables, pixel-perfect alignment, аудит каждого элемента Coverage-first: максимальное покрытие кодовой базы, замена hardcoded → tokens, новые творческие темы
2. Темы 3 темы (NewDark, NewLight, Default) — Figma-источник 5 тем (DarkNeon, Default, Minimalist, Retro, Nature) — собственный дизайн, удалены NewDark/NewLight
3. Layout-токены Максимальная гранулярность: 13 cornerRadii, chatBubbleCorners, sizes, gaps, padding Базовые: 3 cornerRadii (small/medium/large), без sizes/gaps/padding
4. Эффекты GlassEffect → применён к event cards, gradient blur в chat, gradient locations GlassEffect определён, но не применён к UI-компонентам
5. Dead code Не трогали — фокус на добавлении Удалён dead code: SBP flow, inBlackList, Instagram icon, FigmaIcons ассеты
6. Документация Design tracker (GitHub Pages), audit reports, layout tracker TEST_CHECKLIST.md — 81 строка проверок, MIGRATION_QUESTIONS.md
7. CI/CD Без изменений GitHub Actions: SPM cache, TestFlight deploy, build numbers
8. Media Editor Без изменений Полный photo/video editor (фильтры, text overlays, undo, rotation)
